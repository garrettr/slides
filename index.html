PGP (GPG)

  • Widely used for encrypting/signing communications (email)
  • Also widely used for signing software releases (Debian packages, Git tags)

PGP (GPG)

Unfortunately, PGP is hard to use (even for technically capable people)

User errors cause or contribute to most computer security failures, yet user interfaces for security still tend to be clumsy, confusing, or near-nonexistent.

Why Johnny Can't Encrypt: A Usability Evaluation of PGP 5.0 - Usenix

This was published in 1999, but not much has changed.

Tor (and Tor Hidden Services)

  • Tor is an onion routing network designed to anonymize TCP network traffic
  • Widely used for privacy and censorship resistance
  • Tor Hidden services provide anonymous publishing

Tor is the king of high-secure, low-latency anonymity

- The NSA

Tor (and Tor Hidden Services)

  • Unfortunately, Tor can also be somewhat hard to use.
  • Installing Tor standalone
    • 6 minute tutorial video on Tor's site
    • Must use the command line on Mac/Linux
  • Once installed, it's hard to know if your application is using Tor safely (and not, say, leaking all your DNS requests).
  • Tor Project addresses these problems with the Tor Browser Bundle
